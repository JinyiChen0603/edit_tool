## 题目
In the Cartesian coordinate system $xOy$, let $AB$ be a chord of the parabola $y^2 = 4x$ passing through the point $F(1, 0)$. The circumcircle of $\triangle AOB$ intersects the parabola at a point $P$ (distinct from points $O$, $A$, and $B$). If $PF$ bisects $\angle APB$, find all possible values of $|PF|$.

## 解答
1. Geometric Properties and Initial Equations
Let the coordinates of the points be $A\left(\frac{y_1^2}{4}, y_1\right)$, $B\left(\frac{y_2^2}{4}, y_2\right)$, and $P\left(\frac{y_3^2}{4}, y_3\right)$. Given the conditions, $y_1, y_2,$ and $y_3$ are pairwise distinct and non-zero.
Since the chord $AB$ passes through the focus $F(1, 0)$, we can represent the line $AB$ as $x = ty + 1$. Substituting this into the parabola equation $y^2 = 4x$, we get:

$$y^2 - 4ty - 4 = 0$$

According to Vieta's formulas:

$$y_1 y_2 = -4 \quad \cdots\cdots (1)$$
Now, consider the circumcircle of $\triangle AOB$. Since it passes through the origin $O(0,0)$, its equation is $x^2 + y^2 + dx + ey = 0$. Substituting the parabola's property $x = \frac{y^2}{4}$ into the circle's equation:

$$\frac{y^4}{16} + y^2 + \frac{d}{4}y^2 + ey = 0 \implies y^4 + (16 + 4d)y^2 + 16ey = 0$$

The roots of this quartic equation are $y_1, y_2, y_3,$ and $0$. Since the coefficient of the $y^3$ term is zero, the sum of the roots must be zero: $y_1 + y_2 + y_3 + 0 = 0$, which gives:

$$y_3 = -(y_1 + y_2) \quad \cdots\cdots (2)$$
2. Applying the Angle Bisector Theorem
Since $PF$ is the angle bisector of $\angle APB$, the Angle Bisector Theorem states:

$$\frac{|PA|}{|PB|} = \frac{|FA|}{|FB|}$$

Using the focal radius property of a parabola ($|FA| = x_1 + \frac{p}{2} = \frac{y_1^2}{4} + 1$), we have:

$$\frac{|PA|}{|PB|} = \frac{\frac{y_1^2}{4} + 1}{\frac{y_2^2}{4} + 1} = \frac{y_1^2 + 4}{y_2^2 + 4}$$

Substituting $4 = -y_1y_2$ from Eq. (1):

$$\frac{|PA|}{|PB|} = \frac{y_1^2 - y_1y_2}{y_2^2 - y_1y_2} = \frac{y_1(y_1 - y_2)}{y_2(y_2 - y_1)} = \frac{|y_1|}{|y_2|}$$

Squaring both sides yields:

$$\frac{|PA|^2}{|PB|^2} = \frac{y_1^2}{y_2^2} \quad \cdots\cdots (3)$$
3. Solving for the Coordinates
Using the distance formula for $|PA|^2$ and $|PB|^2$ in Eq. (3):

$$\frac{y_1^2}{y_2^2} = \frac{\left(\frac{y_3^2 - y_1^2}{4}\right)^2 + (y_3 - y_1)^2}{\left(\frac{y_3^2 - y_2^2}{4}\right)^2 + (y_3 - y_2)^2}$$

Substitute $y_3 = -(y_1 + y_2)$ from Eq. (2) and $y_1y_2 = -4$ from Eq. (1) into the expression. After simplifying the algebraic terms (as shown in the previous steps):

$$(y_1^2 - y_2^2)(y_1^4 + y_1^2y_2^2 + y_2^4 - 192) = 0$$

If $y_1^2 = y_2^2$, then $y_1 = -y_2$ (since $y_1 \neq y_2$), which leads to $y_3 = 0$ from Eq. (2). This contradicts the condition that $P$ is distinct from $O$. Thus:

$$y_1^4 + y_1^2y_2^2 + y_2^4 = 192$$

Using $(y_1y_2)^2 = (-4)^2 = 16$:

$$(y_1^2 + y_2^2)^2 - y_1^2y_2^2 = 192 \implies (y_1^2 + y_2^2)^2 = 192 + 16 = 208$$

Therefore, $y_1^2 + y_2^2 = \sqrt{208} = 4\sqrt{13}$.
4. Calculating $|PF|$
Finally, we calculate the focal length $|PF|$ using the focal radius formula:

$$|PF| = \frac{y_3^2}{4} + 1 = \frac{(y_1 + y_2)^2 + 4}{4} = \frac{y_1^2 + y_2^2 + 2y_1y_2 + 4}{4}$$

Substituting $y_1^2 + y_2^2 = 4\sqrt{13}$ and $y_1y_2 = -4$:

$$|PF| = \frac{4\sqrt{13} + 2(-4) + 4}{4} = \frac{4\sqrt{13} - 4}{4} = \sqrt{13} - 1$$

$$Answer$$$$\sqrt{13} - 1$$


## 答案
$$\sqrt{13} - 1$$
