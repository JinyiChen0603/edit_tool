## 题目
\section*{铁路高架桥设计问题}

\textbf{问题描述：}

一家土木工程公司正在设计一座由2019个连续编号的支撑桥墩组成的线性铁路高架桥。第 \(i\) 个桥墩的高度用 \(h_i\) 米表示。由于地质和美学限制，高度必须是非递减的整数，从开始的1米到结尾的99米，即 \(1 = h_1 \leq h_2 \leq \cdots \leq h_{2019} = 99\)。

高架桥的总建造成本 \(C\)（以百万美元计）使用以下特定的效率公式计算：
\[ C = \sum_{i=1}^{2019} h_i^2 - \sum_{i=1}^{2017} h_i h_{i+2} \]

这里，第一个求和代表每个单独桥墩的材料成本，它随着高度的平方而增加。第二个求和代表结构稳定性补贴；因为相隔一跨的桥墩提供相互支撑，成本会因它们高度的乘积而降低。

\textbf{问题：}

确定项目的\textbf{最小可能总成本 \(C_0\)}，并计算实现此最小成本的不同高度配置 \((h_1, h_2, \ldots, h_{2019})\) 的数量。


## 解答
\section*{解答}

设 \(a_i = h_i\)。从给定条件，我们有
\[ 2C = a_1^2 + a_2^2 + a_{2018}^2 + a_{2019}^2 + \sum_{i=1}^{2017} (a_{i+2} - a_i)^2. \tag{①} \]

由于 \(a_1, a_2\) 和 \(a_{i+2} - a_i\) （\(i = 1, 2, \dots, 2016\)）都是非负整数，因此 \(a_1^2 \ge a_1,\; a_2^2 \ge a_2\)，且 \((a_{i+2} - a_i)^2 \ge a_{i+2} - a_i\) （\(i = 1, 2, \dots, 2016\)）。

因此
\[ a_1^2 + a_2^2 + \sum_{i=1}^{2016} (a_{i+2} - a_i)^2 \ge a_1 + a_2 + \sum_{i=1}^{2016} (a_{i+2} - a_i) = a_{2017} + a_{2018}. \tag{②} \]


从①和②我们得到
\[ 2C \ge a_{2017} + a_{2018} + (a_{2019} - a_{2017})^2 + a_{2018}^2 + a_{2019}^2, \]

使用 \(a_{2019} = 99\) 以及 \(a_{2018} \ge a_{2017} > 0\) 得到
\[ C \ge \frac{1}{2}\bigl( 2a_{2017} + (99 - a_{2017})^2 + a_{2017}^2 + 99^2 \bigr) = (a_{2017} - 49)^2 + 7400 \ge 7400. \tag{③} \]

另一方面，设
\begin{align*}
&a_1 = a_2 = \cdots = a_{2020} = 1,\\
&a_{2020+12k-1} = a_{2020+12k} = k\quad(k = 1, 2, \dots, 49),\\
&a_{2030} = 99,
\end{align*}

则可以验证上述所有不等式都变为等式；因此 \(C\) 的最小值为 \(C_0 = 7400\)。

\subsection*{等号成立条件}

我们现在考察③中等号成立的条件。等号成立要求 \(a_{2017} = a_{2018} = 49\)，且②中所有不等式都是等式，即 \(a_1 = a_2 = 1\) 且 \(a_{i+2} - a_i \in \{0, 1\}\) （\(i = 1, 2, \dots, 2016\)）。

因此 \(a_1 \le a_2 \le \cdots \le a_{2018} = 49\)，并且对于每个 \(k\) （\(1 \le k \le 49\)），在 \(a_1, a_2, \dots, a_{2018}\) 中至少有两项等于 \(k\)。容易验证这个条件对于③中等号成立也是充分的。

对于每个 \(k\) （\(1 \le k \le 49\)），设 \(a_1, a_2, \dots, a_{2018}\) 中等于 \(k\) 的项数为 \(1 + n_k\)。则每个 \(n_k\) 是正整数，且
\[ \sum_{k=1}^{49} (1 + n_k) = 2018 \quad\Longrightarrow\quad n_1 + n_2 + \cdots + n_{49} = 1969. \]

该方程的正整数解 \((n_1, n_2, \dots, n_{49})\) 的数量为 \(\binom{1968}{48}\)。每个这样的解恰好对应一个使③中等号成立的数组 \((a_1, a_2, \dots, a_{2019})\)。

因此，使 \(C = C_0\) 的数组 \((a_1, a_2, \dots, a_{2019})\) 的数量为 \(\binom{1968}{48}\)。

\vspace{1em}
\textbf{答案：}

     最小成本：\(C_0 = 7400\) 百万美元
     配置数量：\(\binom{1968}{48} = C_{1968}^{48}\)



## 答案
\(\binom{1968}{48} = C_{1968}^{48}\)
