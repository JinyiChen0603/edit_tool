## 题目
Find the smallest positive integer $n$ ($n > 0$) such that $3n + 1$ and $4n + 1$ are both perfect squares.Requirement: Provide the exact value of $n$ and detail the logical deduction involved.

## 解答
Step 1: 建立方程模型
设 $3n + 1 = x^2$ 和 $4n + 1 = y^2$，其中 $x, y$ 为正整数。
为了消去 $n$，我们利用这两个方程构造一个新的关系式：

$$4(3n + 1) = 4x^2$$
$$3(4n + 1) = 3y^2$$
相减得：

$$4x^2 - 3y^2 = 4(3n+1) - 3(4n+1)$$
$$4x^2 - 3y^2 = 12n + 4 - 12n - 3$$
$$4x^2 - 3y^2 = 1$$

Step 2: 转化为标准佩尔方程
方程 $4x^2 - 3y^2 = 1$ 看起来像佩尔方程，但 $x^2$ 的系数是 4。
我们可以通过换元将其标准化。
令 $X = 2x$，则方程变为：

$$X^2 - 3y^2 = 1$$
这是一个标准的佩尔方程 $X^2 - Dy^2 = 1$，其中 $D=3$。
注意隐形约束： 由于 $X = 2x$，因此佩尔方程的解 $X$ 必须是偶数。如果算出的 $X$ 是奇数，则该解无效（无法还原为整数 $x$）。

Step 3: 求解佩尔方程 $X^2 - 3y^2 = 1$
我们需要找到 $\sqrt{3}$ 的连分数展开来确定基本解。

$$\sqrt{3} \approx 1.732 \implies [1; \overline{1, 2}]$$

渐进分数序列：
	0.	$1/1 \implies 1^2 - 3(1)^2 = -2$ (非解)
	0.	$2/1 \implies 2^2 - 3(1)^2 = 1$ (基本解)
因此，基本解（Fundamental Solution）为 $(X_1, y_1) = (2, 1)$。
所有正整数解 $(X_k, y_k)$ 可以通过基本单位 $2 + \sqrt{3}$ 的幂次生成：

$$X_k + y_k\sqrt{3} = (2 + \sqrt{3})^k$$

Step 4: 筛选满足偶数约束的解
我们需要检查每一个 $X_k$，看它是否为偶数，从而求出 $x$，进而求出 $n$。
Case $k=1$ (基本解):

$$X_1 + y_1\sqrt{3} = (2 + \sqrt{3})^1 = 2 + \sqrt{3}$$
	•	$X_1 = 2$ (偶数，符合条件)
	•	$x = X_1 / 2 = 1$
	•	代回求 $n$：$3n + 1 = x^2 = 1^2 \implies 3n = 0 \implies n = 0$。
	•	排除： 题目要求 $n$ 为正整数 ($n > 0$)。
Case $k=2$:

$$X_2 + y_2\sqrt{3} = (2 + \sqrt{3})^2 = 4 + 3 + 4\sqrt{3} = 7 + 4\sqrt{3}$$
	•	$X_2 = 7$
	•	排除： $X$ 必须是偶数，7 是奇数，无法得到整数 $x$。
Case $k=3$:

$$X_3 + y_3\sqrt{3} = (2 + \sqrt{3})^3 = (2 + \sqrt{3})(7 + 4\sqrt{3})$$
$$= 14 + 8\sqrt{3} + 7\sqrt{3} + 12$$
$$= 26 + 15\sqrt{3}$$
	•	$X_3 = 26$ (偶数，符合条件！)
	•	$x = X_3 / 2 = 13$

Step 5: 最终计算与验证
我们找到了满足条件的最小 $x = 13$。现在求 $n$：

$$3n + 1 = x^2 = 13^2 = 169$$
$$3n = 168$$
$$n = 56$$

验证第二个条件：
$$4n + 1 = 4(56) + 1 = 224 + 1 = 225$$
$$225 = 15^2$$

## 答案
56
