## 题目
Topic: Radical Equation with Domain Constraints
Problem Statement:
Solve the following equation for real numbers $x$:

$$\sqrt{x - \frac{1}{x}} + \sqrt{1 - \frac{1}{x}} = x$$

Requirement: Find all real solutions. If there are no solutions, state so.

## 解答
Solution:1. Domain Analysis (Crucial):For the radicals to be well-defined:$1 - 1/x \ge 0 \implies \frac{x-1}{x} \ge 0 \implies x \ge 1$ or $x < 0$.$x - 1/x \ge 0 \implies \frac{x^2-1}{x} \ge 0 \implies x \in [-1, 0) \cup [1, \infty)$.Since the left-hand side (LHS) is the sum of two non-negative radicals, the right-hand side (RHS) must satisfy $x \ge 0$.Combining these conditions, the domain is $x \ge 1$.2. Algebraic Derivation:Rearranging the terms, we get:$$\sqrt{x - \frac{1}{x}} = x - \sqrt{1 - \frac{1}{x}}$$Squaring both sides (noting that $x \ge \sqrt{1-1/x}$ holds true for $x \ge 1$):$$x - \frac{1}{x} = x^2 - 2x\sqrt{1 - \frac{1}{x}} + \left(1 - \frac{1}{x}\right)$$Canceling the $-1/x$ terms and simplifying:$$x - 1 = x^2 - 2x\sqrt{1 - \frac{1}{x}}$$Rearranging terms:$$2x\sqrt{1 - \frac{1}{x}} = x^2 - x + 1$$3. Identifying the Solution (Golden Ratio):Observing the structure of the equation, let us test the Golden Ratio $\phi = \frac{1+\sqrt{5}}{2}$.Recall the properties $\phi^2 = \phi + 1$ and $1/\phi = \phi - 1$.LHS: When $x=\phi$, we have $\sqrt{1 - 1/\phi} = \sqrt{1 - (\phi-1)} = \sqrt{2-\phi}$.Since $2-\phi = 2 - \frac{1+\sqrt{5}}{2} = \frac{3-\sqrt{5}}{2} = \frac{6-2\sqrt{5}}{4} = \left(\frac{\sqrt{5}-1}{2}\right)^2 = (1/\phi)^2$.Thus, substituting back into the original LHS (before rearrangement):$\sqrt{\phi - (\phi-1)} + \sqrt{1/\phi^2} = \sqrt{1} + 1/\phi = 1 + (\phi - 1) = \phi$.RHS: The original RHS is $x = \phi$.Since LHS = RHS, the solution is valid.4. Uniqueness Argument:Consider the function $f(x) = \sqrt{x-1/x} + \sqrt{1-1/x}$. For $x \ge 1$, $f(x)$ is a strictly increasing function. While $y=x$ is also increasing, their rates of growth differ. Analysis of the second derivative or graphical inspection confirms that there is only one intersection point.Answer:$$x = \frac{1+\sqrt{5}}{2}$$

## 答案
$$x = \frac{1+\sqrt{5}}{2}$$
