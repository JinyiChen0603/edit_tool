## 题目
设点集 $X = \{x_1, x_2, \dots, x_n\} \subset \mathbb{R}$ 的直径 $\text{diam}(X) \leq 1$。定义系统的平均相互作用势为 $\Phi = \frac{1}{n^2} \sum_{1 \leq i < j \leq n} (x_i - x_j)^2$，求 $\Phi$ 的最大值。

## 解答
1. Let the mean of these $n$ points be $\bar{x} = \frac{1}{n}\sum_{i=1}^n x_i$.  
Consider the numerator of $\Phi$:  
$$ \sum_{1 \leq i < j \leq n} (x_i - x_j)^2 $$  
Using the identity $\sum_{i<j} (x_i - x_j)^2 = n \sum_{i=1}^n (x_i - \bar{x})^2$, we transform it into the variance form:  
$$ \Phi = \frac{1}{n^2} \cdot n \sum_{i=1}^n (x_i - \bar{x})^2 = \frac{1}{n} \sum_{i=1}^n (x_i - \bar{x})^2 = \mathrm{Var}(X) $$

2. Utilizing the upper bound property of variance  
Let the minimum value of the set $X$ be $a$ and the maximum value be $b$. From the definition of the diameter, we know that $b - a \leq 1$.  
Without loss of generality, assume all points lie within the interval $[a, b]$. For a random variable taking values in the interval $[a, b]$, there is a definite upper bound for its variance:  
$$ \mathrm{Var}(X) \le \frac{(b-a)^2}{4} $$  
For any real number $t$, we have $(x_i - t)^2 \ge 0$.  
The variance can be expressed as:  
$$
\begin{aligned}
\mathrm{Var}(X) &= \frac{1}{n} \sum (x_i - \bar{x})^2 \\
&= \frac{1}{n} \sum \left[ (x_i - \frac{a+b}{2}) + (\frac{a+b}{2} - \bar{x}) \right]^2 \\
&= \frac{1}{n} \sum (x_i - \frac{a+b}{2})^2 + \left(\frac{a+b}{2} - \bar{x}\right)^2 + \frac{2}{n}\left(\frac{a+b}{2} - \bar{x}\right) \sum \left(x_i - \frac{a+b}{2}\right)
\end{aligned}
$$
Since $\bar{x}$ is the mean, the cross term $\sum (x_i - \frac{a+b}{2}) = n(\bar{x} - \frac{a+b}{2})$, so the last two terms combine to $-\left(\frac{a+b}{2} - \bar{x}\right)^2$.  
Therefore:  
$$ \mathrm{Var}(X) = \frac{1}{n} \sum (x_i - \frac{a+b}{2})^2 - \left(\frac{a+b}{2} - \bar{x}\right)^2 \le \frac{1}{n} \sum (x_i - \frac{a+b}{2})^2 $$
Since $x_i \in [a, b]$, it follows that $|x_i - \frac{a+b}{2}| \le \frac{b-a}{2}$, which implies $(x_i - \frac{a+b}{2})^2 \le \left(\frac{b-a}{2}\right)^2$. Substituting this in, we get:  
$$ \mathrm{Var}(X) \le \frac{1}{n} \cdot n \cdot \left(\frac{b-a}{2}\right)^2 = \frac{(b-a)^2}{4} $$

3. Calculating the maximum value  
From the problem condition $\mathrm{diam}(X) \le 1$, i.e., $b - a \le 1$, substitute this into the inequality above:  
$$ \Phi = \mathrm{Var}(X) \le \frac{1^2}{4} = \frac{1}{4} $$  
When half of the elements in the set $X$ take the value $a$ and the other half take the value $a+1$ (i.e., distributed at both ends of the interval), the variance reaches the maximum value of $1/4$.  
Therefore, the maximum value of $\Phi$ is **1/4**.

## 答案
$$\frac{1}{4}$$
